# 🎮 Tahmin Oyunu Platformu

Bu proje PHP Programlama dersi kapsamında geliştirilmiş eğlence amaçlı bir web uygulamasıdır.  
Kullanıcılar sisteme kayıt olup giriş yapabilir, oyun oynayabilir ve skor kazanabilirler.

---

# 🚀 Proje Özellikleri

- 👤 Kullanıcı kayıt sistemi
- 🔐 Kullanıcı giriş sistemi
- 🎯 Sayı tahmin oyunu
- 🧠 Kelime tahmin oyunu
- 🏆 Skor sistemi
- 📊 Leaderboard (sıralama tablosu)
- 💾 MySQL veritabanı kullanımı
- 🎨 Bootstrap ile modern tasarım

---

# 🛠 Kullanılan Teknolojiler

- PHP
- MySQL
- XAMPP
- Bootstrap
- GitHub

---

# 🎯 Oyunlar

## 🎯 Sayı Tahmin Oyunu
Sistem 1 ile 100 arasında rastgele sayı üretir.  
Kullanıcı doğru sayıyı tahmin etmeye çalışır.  
Yanlış tahminlerde puan azalır.

---

## 🧠 Kelime Tahmin Oyunu
Sistem rastgele bir kelime seçer ve harflerini karıştırır.  
Kullanıcı doğru kelimeyi tahmin etmeye çalışır.

---

# 🏆 Skor Sistemi

Kullanıcıların puanları veritabanında saklanır.  
Leaderboard ekranında oyuncular sıralanır.

---

# 👥 Grup Projesi

Bu proje grup çalışması olarak geliştirilmiştir.

---

# 📌 Kurulum

1. XAMPP kurulu olmalıdır.
2. Proje dosyaları `htdocs` klasörüne eklenmelidir.
3. MySQL üzerinde `tahmin_oyunu` veritabanı oluşturulmalıdır.
4. Tarayıcıdan proje çalıştırılabilir.
